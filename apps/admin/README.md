# Admin — Directus

Directus запускается как Docker-сервис `directus`.

- URL: `http://localhost:8055` (или через Cloudflare Tunnel)
- Админка на русском языке
- Коллекции управляются через веб-интерфейс Directus

## Collections

Предустановленные коллекции (создаются вручную при первом запуске):

- Товары (products)
- Категории (categories)
- Теги (tags)
- Заказы (orders)
- Пользователи (users)
- Покупки (orders + order_items)
- Промокоды (promo_codes)
- Поддержка (support_tickets)
- Рассылки (notifications)
- Аналитика (views через directus insights)
- Настройки (settings)
- Админы (admins)
- Логи админов (admin_logs)
- Отправленные ссылки (link_delivery_logs)

## Переводы

- Directus Collections: русский
- Mini App: en, ru, de, el, ro, bg, mo, sr, tr
- Support: только английский
- Receipt: язык пользователя
